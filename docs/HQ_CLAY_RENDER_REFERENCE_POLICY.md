# HQ Clay Render Reference Policy

Shape Lab currently targets untextured mesh quality.

Product-truth previews:

- clay renders
- matcap-style renders
- studio renders produced from Shape Lab mesh output
- wireframe and silhouette views
- contact sheets generated from Shape Lab outputs

Not product truth:

- photoreal renders
- textured screenshots
- painted concept art
- environment beauty shots
- reference boards
- marketing mockups

Private reference boards may guide author taste, but they must not be marketed
as reconstruction, IP targeting, or output parity.

Current non-claims:

- no UV promise
- no materials promise
- no texture promise
- no rigging promise
- no animation promise
- no marketplace-ready game-asset promise
- no photoreal output promise
