# HQ Mesh Validation Checklist

HQ review evaluates untextured mesh quality, not materials or renderer polish.

Required automatic evidence:

- profile compiles through the Foundry compiler
- model validation has no error-severity issues
- triangle count and triangle budget are recorded
- semantic part inventory is recorded
- required role coverage is recorded
- provider/attachment validity is recorded
- candidate survival count is recorded
- primary controls have visible difference evidence
- export/reopen status is recorded

Required manual review for Usable or Showcase:

- silhouette reads clearly at thumbnail size
- proportions and repeated elements look intentional
- component boundaries are understandable
- no obvious procedural artifacts dominate the model
- style/provider choices remain coherent
- no Advanced Recipe knowledge is needed for the intended novice task

Unsupported current outputs:

- UV layout
- materials
- textures
- rigging
- animation
- photoreal output
- marketplace package publishing
