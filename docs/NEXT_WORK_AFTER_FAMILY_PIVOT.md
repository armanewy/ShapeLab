# Next Work After Family Pivot

Date: 2026-06-30

The family pivot is superseded by the Primitive Direct-Make reset. The immediate
work is no longer another named object feature. The product must first make
primitive editing simple, direct, and trustworthy.

## New Direction

```text
Direct primitive editing first.
Composition second.
Reusable kit or family authoring third.
LLM assistance later, only as planner or repair help.
```

## Serial Milestones

Use one visible operation per milestone:

1. Primitive Direct-Make Product Reset
2. Retire Active Variation UI
3. Primitive Property Schema Contracts
4. Box and Flat Panel Direct Property UI
5. Sphere Primitive Direct Controls
6. Primitive Composition Contracts
7. Panel + Knob Composition Prototype

The visible user sequence is:

```text
direct-edit Box
-> direct-edit Flat Panel
-> direct-edit Sphere
-> make knob-like form from Sphere
-> attach knob-like form to panel through safe anchor
```

## Preferred Next Step

Remove generated variation behavior from the active primitive workflow. Do not
delete every candidate-related backend type yet. Mark candidate and variation
machinery as internal, legacy, or test-only until deterministic presets or
suggestions are intentionally reintroduced.

## Continuing Criteria

- The user edits exact primitive properties, not opaque generated ideas.
- Primitive properties are typed, bounded, and validated.
- Direct sliders are allowed only for geometry-preserving edits.
- Topology-changing choices must be discrete.
- No UI copy claims surface/material, UV, rigging, animation, runtime LLM, or
  game-ready support.
- Family Studio Lite remains paused until direct primitive and composition
  flows are stable.

## Blocked

- Further Door Handle / Knob feature work before direct primitives are stable.
- Door naming before visible door cues pass a later gate.
- Material looks, UV/texturing, rigging, animation, or game-ready UI.
- Runtime LLM integration.
- Full Family Studio public flow.
- Base-family field sprawl such as optional handles, vents, materials, or rigs
  on Box Primitive.
