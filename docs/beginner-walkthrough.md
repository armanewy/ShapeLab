# Shape Lab Beginner Walkthrough

This walkthrough uses only built-in presets and the visible app controls.

## Start

1. Open `New From Preset` and choose `Sky Shrine`.
2. In `Parts`, choose `Floating halo ring` or any part tagged `#focus`.
3. In `Options`, leave `What can change next` on `This part`.
4. Turn on only the value groups you want to try first, such as `Outline` and `Position`.

## Generation 1

1. Choose `Refine` for small changes.
2. Press `Generate Options`.
3. Compare each option card with the `Current model` control card.
4. Read `What changes` to see which part and value will change.
5. Press `Choose This Option` on the card you prefer.

## Generation 2

1. Pick a different focus part, such as `Lifted roof slab`.
2. Switch to `Explore` for bigger alternatives.
3. Press `Generate Options`.
4. Choose one option to create the next history step.

## Generation 3

1. Pick `Everything` if the whole preset should be allowed to change.
2. Use `Keep` beside any value that must stay fixed.
3. Press `Generate Options`.
4. Choose one option.

## Make A Branch

1. Open `History`.
2. Press `Undo` or click an earlier step in `Path from the start`.
3. Generate options again from that earlier step.
4. Choose a different option.

The earlier step now has a branch point: one path follows the first option you chose, and the other path follows the newer option.

## Reset

Use `Reset to Preset` in the inspector when you want to return to the original built-in preset and clear the generated options.
